Frequently Asked Questions

Have questions? We've got answers. If you can't find what you're looking for, feel free to contact us.

Apakah server langsung aktif setelah pembayaran?

Ya, semua layanan kami menggunakan sistem instant setup. Begitu pembayaran terverifikasi, layanan langsung aktif dan siap digunakan.

Bagaimana cara mengelola server saya?

Kami menyediakan kontrol panel modern (cPanel untuk web hosting, Pterodactyl Custom untuk game server) yang mudah digunakan bahkan untuk pemula.

Bisa upgrade resource di tengah jalan?

Ya, resource server seperti CPU, RAM, dan storage dapat di-upgrade kapan saja tanpa downtime.

Dimana lokasi datacenter Skyes Hosting?

Kami memiliki datacenter di Indonesia dengan koneksi langsung ke Telkom Eyeball serta jaringan global untuk latency rendah.

Bagaimana cara menghubungi support?

Tim support kami tersedia melalui Tiket, Discord, dan WhatsApp untuk membantu kapan saja.

Apakah ada fitur Backup?

Ya, backup bisa dilakukan secara manual atau dengan Schedule di Control Panel.

Bagaimana cara migrasi ke Skyes?

Kamu bisa menggunakan Server Importer di panel kami atau upload manual servernya lewat SFTP.

Seberapa aman data saya di Skyes?

Kami memiliki backup harian on-site dan per 3 hari off-site, namun user tetap disarankan untuk memiliki backup sendiri. Backup kami hanya digunakan jika ada masalah dari pihak Skyes seperti kerusakan hardware.
